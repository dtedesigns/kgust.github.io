Browser Configuration

You must enable Experimental Extension APIs before installing the Extension.

  • Go to chrome://flags.
  • Find Experimental Extension APIs.
  • Click "Enable".
  • Restart your browser.

Install the AutoSave Extension.

  • Install AutoSave (Direct link)

Configure your routes:

  • Go to Tools->Extensions
  • Under DevTools autosave, click "options".
  • Set up a route. E.g.:
    • Resource: ^http://localhost/
    • Post to: ^http://127.0.0.1:9104/save

Now you are all ready to go!
