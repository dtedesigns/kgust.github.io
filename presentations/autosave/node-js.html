Node.js and AutoSave Server

Installing the AutoSave Service

  1. Installing the AutoSave service is pretty easy. The service itself is very light.
    • On Windows or Mac, just go to nodejs.org and click "Download".
    • On Linux systems just use your native package manager.

  2. Then fire up your favorite terminal application, and type:

    3. npm install -g autosave
